# Simulator projects

LittlevGL can run on PC to make the development easier and faster. You can choose from the following ready to use projects:

- [lv_sim_eclipse_sdl](https://github.com/littlevgl/lv_sim_eclipse_sdl): Eclipse based simulator project. Easy to use on Linux and OSX.
- [lv_sim_codeblock_win](https://github.com/littlevgl/lv_sim_codeblocks_win): Simulator in CodeBlocks. Works out-of-the-box on Windows.
- [lv_sim_visual_studio_sdl](https://github.com/littlevgl/lv_sim_visual_studio_sdl): Visual Studio Project. Works out-of-the-box on Windows.
- [lv_sim_emscripten](https://github.com/littlevgl/lv_sim_emscripten): Compiles JavaScript files from C to run the UI in browser.
- [lv_platformio](https://github.com/littlevgl/lv_platformio): Project in PlatformIO which has simulator and embedded hardware configuration options.
